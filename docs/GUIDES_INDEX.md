## Quick Start

1. **Read the universal guide first**: [GETTING_STARTED.md](GETTING_STARTED.md)
2. **Follow your platform guide**: Choose the appropriate language-specific guide
3. **Configure your environment**: Set up models and binaries as described
4. **Test your installation**: Run the provided examples

Each guide includes:
- Prerequisites and installation
- Environment setup
- Configuration examples
- Sample code and testing
- Common pitfalls and solutions
- Performance recommendations

Start with the universal guide to understand the overall architecture, then dive into your specific platform documentation.
